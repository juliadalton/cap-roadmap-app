# Release Notes - February 22, 2026 (Release 2)

## Summary
This release adds a view toggle to the Technical Integration page, allowing users to switch between grouping by Acquisition or by Project.

---

## Updated Pages

### Technical Integration (`/technical-integration`)

**New View Toggle Feature**
- Added "Group by" toggle above the Gantt chart
- Two view modes available:
  - **Acquisition View** (default): Groups by acquisition with projects nested below
  - **Project View**: Groups by project with acquisitions nested below

**Acquisition View (existing behavior)**
- Acquisition displayed as parent row with color indicator
- Projects nested below each acquisition
- Timeline spans showing overall acquisition duration
- Project bars showing individual project timelines

**Project View (new)**
- Project displayed as parent row with folder icon
- Acquisitions nested below each project
- Click project name to view details modal
- Timeline span showing project duration
- Dot indicators for each associated acquisition
- Edit/delete controls available for editors

**UI Details**
- Toggle hidden during PDF export (uses currently selected view)
- Expand/collapse functionality works in both views
- Editor controls (add, edit, delete) work in both views
- Consistent styling between both view modes

---

## Technical Changes

### State Management
- Added `viewMode` state: `'acquisition' | 'project'`
- Added `expandedProjects` state for project view expansion tracking

### New Helper Functions
- `sortedProjects`: Alphabetically sorted projects list
- `getProjectAcquisitions`: Gets acquisitions linked to a project
- `getProjectTimelineSpan`: Calculates timeline span for a project
- `getProjectColor`: Determines color based on linked acquisitions
- `toggleProjectExpansion`: Handles expand/collapse for project rows
