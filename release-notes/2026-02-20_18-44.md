# Release Notes - February 20, 2026

## Summary
This release introduces the Company Integrations feature set, providing comprehensive acquisition tracking and integration progress visualization tools.

---

## New Pages

### Acquisition Tracker (`/acquisition-tracker`)
A progress tracking dashboard for monitoring acquisition integrations with:
- **Dual-track progress visualization**: Technical Integration and Client Migrations tracks
- **Overall integration progress bar** combining both tracks
- **Technical Integration metrics**:
  - Dev Platform connection status (Connected/Not Connected)
  - Functionality in Console: Multi-segment progress bar showing completed (green), in-progress (blue), and to-do (gray) epics
- **Client Migration metrics**:
  - Clients With Access to Console
  - Clients Active in the Console
- **Expandable acquisition cards** with integration overview
- **Search and sort controls** (A-Z/Z-A by name)
- **Anchor linking** from Acquisition List page to specific acquisitions

### Technical Integration (`/technical-integration`)
A Gantt-style timeline view showing:
- Acquisitions with their associated projects on a milestone-based timeline
- Color-coded project bars matching acquisition colors
- Clickable project names opening a view-only modal with:
  - Project description
  - Start/End milestones
  - Relevant links
  - Associated acquisitions
  - Associated functionality epics
- Expandable/collapsible acquisition rows

### Acquisition List (`/acquisitions`)
Updated with:
- Alphabetical sorting (A-Z)
- Clickable project count opening a modal with full project details
- "View in Tracker" links that anchor to the specific acquisition on the Tracker page
- Acquisition color displayed on the building icon

## New Features

### Acquisition Color Picker
- Acquisitions now have a customizable display color
- Color picker in the acquisition editor with:
  - 12 preset color swatches
  - Native color picker for custom colors
  - Hex code text input
- Color used consistently across:
  - Acquisition List page (building icon)
  - Acquisition Tracker page (building icon)
  - Technical Integration page (timeline bars, project spans, indicators)

### Projects Modal
- View-only modal showing project details
- Displays: description, milestones, relevant links, associated epics
- Scrollable with horizontal rules separating multiple projects

## Database Schema Changes

### New Tables
- **AcquisitionProgress**: Tracks integration progress metrics per acquisition
  - `disposition`: Integration type (FULL, FEATURE_LAYER, SUNSET)
  - `devPlatform`: Boolean for platform connection status
  - `functionalityEpicsToDo/InProgress/Complete`: Epic counts
  - `clientCountTotal/AccessCount/ActiveCount`: Client metrics

- **FunctionalityEpic**: Tracks individual functionality epics
  - `epicName`, `epicLink`, `status` (TO_DO, IN_PROGRESS, COMPLETE)
  - Links to both Acquisition and Project tables

- **AcquisitionClientCount**: Tracks individual client records
  - `clientVitallyId`, `clientName`, `activeInConsole`

### Model Updates
- **Acquisition**: Added `color` field for display customization
- **Project**: Added `epics` relation to FunctionalityEpic
